> >[!Example]- Волкодав *(серия книг)*
> > Если рассматривать книги не по хронологии их написания и издания, а по развитию сюжета, то правильным будет порядок, при котором книга «Истовик-камень» стоит на первом месте, а «Мир по дороге» — на втором.
> > 1. [ ] $1995$ — «Волкодав»
> > 2. [ ] $1996$ — «Волкодав. Право на поединок»
> > 3. [ ] $2000$ — «Волкодав. Истовик-камень»
> > 4. [ ] $2003$ — «Волкодав. Знамение пути»
> > 5. [ ] $2003$ — «Волкодав. Самоцветные горы»
> > 6. [ ] $2014$ — «Волкодав. Мир по дороге»
